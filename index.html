Reliable Parkinson’s Disease Detection from EEG using Strict Subject-Wise Validation

"Hypothesis/Project Statement: Subject-independent validation using LOSO provides realistic performance estimates for EEG-based Parkinson’s detection compared to inflated subject-dependent cross-validation."

Problem Statement

Most EEG-based Parkinson’s disease detection studies report extremely high accuracy due to subject-dependent cross-validation, which causes data leakage. Such evaluation strategies lead to inflated performance estimates and poor generalization to unseen patients. There is a critical need for strict subject-wise validation to obtain realistic clinical performance.

Literature Review / Market Research

Recent deep learning approaches including CNNs, GNNs, and time-frequency models report accuracies exceeding 95% using window-wise cross-validation. However, studies applying strict Leave-One-Subject-Out (LOSO) validation report significantly lower but realistic performance (~63–75%), highlighting methodological weaknesses in existing literature.

Research Gap / Innovation

The major gap in existing research is the lack of strict subject-level validation. Most studies optimize architectural complexity while ignoring validation rigor. Our work emphasizes methodological discipline using Leave-One-Subject-Out validation to ensure real-world reliability.
